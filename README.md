# 🖐️ Hand Gesture Recognition (Pattern Recognition)

This project implements a hand gesture recognition system using classical Pattern Recognition techniques.

## 🔍 Methodology
- Image Preprocessing
- HOG Feature Extraction
- PCA for Dimensionality Reduction
- Random Forest Classifier

## 📊 Performance
- Accuracy: ~70–75% on HG14 dataset

## ⚠️ Limitations
- Works best on static images
- Sensitive to lighting and background
- Not optimized for real-time webcam input

## 🚀 Deployment
Deployed using Streamlit Cloud.
